Unlocking the Power of GitHub: A Guide to Open Source Collaboration

Eneko Urunuela, Basque Center on Cognition, Brain and Language, ES

This tutorial provides a comprehensive introduction to using GitHub for collaborative open source development. It covers key concepts, techniques, and best practices for effective collaboration, version control, issue tracking, pull requests, and more. Attendees will gain practical knowledge to confidently contribute and collaborate on open source projects using GitHub.
